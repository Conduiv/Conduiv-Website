Azure Migration — from legacy on-prem to cloud-native

Led the full migration of a regional business unit's core IT infrastructure from aging on-premises servers to Microsoft Azure. Re-platformed Windows Server, Linux hosts, Oracle, and line-of-business applications; replaced Active Directory with Entra ID; migrated file shares to Azure Files and NAS.

  • Retired end-of-life hardware, closed compliance and DR gaps in one cycle.
  • Rolled out Intune for device and application deployment across the fleet.
  • Automated user profile migration to OneDrive via PowerShell and Graph.

2025 · Client Network Upgrades

3G → 4G/LTE cellular migration and NMS cloud cutover

Spearheaded migration of the Network Management System from legacy infrastructure to the Microhard IoT Portal (NMS 2.0) — enabling centralized, cloud-based monitoring and secure remote management for all cellular IoT assets. Upgraded client metering and control sites from 3G to 4G/LTE ahead of carrier sunset.

  • Deployed high-gain outdoor antennas and boosters for challenging remote sites.
  • Coordinated modem swaps, firmware updates, and field-technician troubleshooting.
  • Delivered seamless transitions with minimal downtime on live client networks.
Zero carrier-sunset outages Remote mgmt. unlocked

  • Same approach now applied to internal tooling, runbooks, and onboarding documentation in my day-to-day role.
